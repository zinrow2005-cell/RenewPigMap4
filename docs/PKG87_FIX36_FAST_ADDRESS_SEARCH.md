## 修正重點

1. 上方搜尋列改為「快速地址搜尋」：優先解析台灣地址中的縣市、鄉鎮、村里、路名、段、巷、弄與門牌號碼。
2. 線上搜尋由原本候選逐筆查詢，改為少量精準候選並行查詢，並加入 sessionStorage 快取，重複搜尋會更快。
3. 搜尋結果排序改為：門牌 / 建物優先，其次路名 / 道路，再來行政地名與一般地標。
